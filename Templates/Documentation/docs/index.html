Kular - Digital Responsive OpenCart Theme

Thank you for purchasing our theme.

Kular is a premium theme optimized for creating OpenCart online stores with fully responsive and extremely customization. This theme has powerful control panel with many options to custom your site follow your idea.

Kular theme is fully compatible with OpenCart 3.x

This theme is built based on OpenCart eCommerce CMS. When you use this theme, you should research and have knowledge about OpenCart.

Theme Installation

There are 2 ways to install our theme.

  1. Install Template on default OpenCart store - These steps you need to do:
    1. Install the fresh OpenCart version 3.x on your site
    2. Install template by using admin dashboard
  2. Install QuickStart directly to your site - You can install theme directly on your site like as installing default OpenCart store.

Tutorial Video of Installation

This is video to guide you for installing Kular theme. Hope it help you to make the installation easier.

Install Template

  1. Install default OpenCart 3.x

    Create your database on your server (Example: database in localhost)

    Download OpenCart 3.x package on OpenCart Official Site

    You can get zip file 3.0.2.0-OpenCart.zip when file have been downloaded successfully.

    Extract this file and you have folders, files:

    Click upload folder, you can see the core folders, files of OpenCart eCommerce CMS

    Copy or Upload all files and folders to your server, folder of your site or localhost folder.

    Rename 2 files:

    • config-dist.php -> config.php

    • admin/config-dist.php to admin/config.php

    Now, open browser and load your site (where OpenCart files, folders uploaded or copied) to run OpenCart Installer.

    Example: My url is localhost/dv_kular.

    Click Continue.

    Check all status of your server configuration.

    If OK, Click Continue.

    Fill info of your site correctly, Database is a new database you have created.

    When you done, click Continue.

    Now, your site has been installed OpenCart successfully.

    You can navigate Administration or Frontend.

    You should delete install folder on your site.

    This is OpenCart default Online Store.

    This is a Dashboard when you login admin account.

    Now you have installed default OpenCart successfully, follow next steps to install Kular template.

  2. Install KULAR template

    Extract the file dv_kular_theme_1.0.zip that you downloaded from your items on ThemeForest.

    anh cua folder package

    Now, you can see the zip file of template: kular_template.zip

    Extract this file to get kular_template folder. Access into kular_template folder.

    Copy or Upload all files and folders to OpenCart site that you have installed (merge them with the existing ones).

    NOTE: DON'T Delete or Change anything about "divadata" folder.

    Open browser and navigate to Dashboard.

    Go to Extensions -> Extensions

    Choose Modules

    Click Install to install DIVA Control Panel

    After install module, the notification will appear. Now, click Edit to access DIVA Control Panel

    When DIVA Control Panel show, click Advance

    Choose layout you want to use and click Import

    Go to Extensions -> Modification

    Click Refresh button

    Your site has been installed Kular template and you can see your Frontend.

    Now, you can configure your data by using the OpenCart default methods and Theme Features & Modules

Features & Modules

Kular theme has many modules, features & options (along with OpenCart features) to make you can custom your site follow your idea.

Kular theme has many options to configure category page at Control Panel's Category Page.

Product

You can research and read more about Product at link: http://docs.opencart.com/en-gb/catalog/product/

When your store has products, it has some kinds of products:

  • Featured Products

    chi sang featured module

  • Most Viewed Products

    Most Viewed Products are products which have most viewed on your store. The viewed is calculated by default OpenCart.

    chi sang most view module

  • Bestseller Products

    Bestseller are the most sold products. The sold number is calculated by default OpenCart.

    chi sang module bestseller

  • Latest Products

    Latest Products are the newest products. The newest products are the most recently created products.

    chi sang module latest

  • Special Products

    Special Products are products have special price. You can set special price at Special Tab in product management.

    You can read more at http://docs.opencart.com/en-gb/catalog/product/special/

    chi sang module Special Product

  • Random Products

    Random Products are products that are get randomly. This feature is made by our theme.

    chi sang module random

Kular theme has many options to configure product detail page at Control Panel's Product Detail.

Layout

This is default feature of OpenCart. You can assign your modules to any position of specified layouts (page).

You can know more about Layout at link: http://docs.opencart.com/en-gb/system/design/layout/

Kular theme has 2 layouts for Home page. You can change layout just one click by using Theme Control Panel

Also, this theme has some modules and features to edit layout easily: DIVA Layout Composer and Theme Module Customization

Product Detail

You can configure settings for Product Page.

  1. Related Products: Enable / Disable Related Products in Product Page.

  2. Social Icons: Enable / Disable Social links in Product Page.

  3. Price Tax: Enable / Disable product's price tax.

  4. Product Tags: Enable / Disable product's tags.

  5. Image Zoom Settings

    • Image Zoom: Enable / Disable Image Zoom.
    • Type: Set Zoom's type is Inner or Outer
    • Space: When zoom's type is "Outer", set space between Thumbnail Image and Zoom Image
    • Zoom Title: Enable / Disable product name on zoom image.

  6. Color Swatches Settings:

    These options are used for Products that have options (the options must have type "Select" or "Radio").

    Configure Options: Go to Admin -> Catalog -> Options

    You can add new options or edit existed options.

    This is the Options form

    You should configure option by fill your data:

    • Option Name: Set name of Option.

    • Type: You must choose Select or Radio to use it for Color Swatches.

    • Sort Order: Set order of option.

    • Option Values: Set values of this options.
